## [v0.1.0]

### Added
- https://github.com/src-d/go-errors fork.
- `HttpError` struct to represent http errors.
- Built-in HTTP Bad request error.
- Built-in HTTP Internal server error.
- Built-in HTTP Bad gateway error.
- Built-in HTTP Unauthorized error.
- Built-in HTTP Forbidden error.
- Built-in HTTP Gateway timeout error.
- Built-in HTTP Not found error.
